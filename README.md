# My Ruby Project

## Overview
My Ruby Project is a simple Ruby application designed to demonstrate the structure and functionality of a Ruby project. It includes core classes and methods, an interactive console, and a suite of tests to ensure reliability.

## Installation
To install the necessary dependencies, run the following command:

```
bundle install
```

## Usage
To start an interactive console session, use the following command:

```
bin/console
```

This will allow you to interact with the core functionality of the project.

## Running Tests
To run the test suite, you can use the following command:

```
rspec spec/my_ruby_project_spec.rb
```
